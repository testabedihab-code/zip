# طلبك تم — منصة إعلانات جبلة

منصة إعلانات مبوبة في مدينة جبلة — شقق وسيارات ومعدات للإيجار والبيع.

## الرفع على GitHub Pages

1. Settings → Pages → Source: **Deploy from a branch** → **main** → **/ (root)**
2. انتظر دقيقة — الموقع جاهز

## الملفات

| الملف | الوصف |
|-------|-------|
| `index.html` | الهيكل HTML |
| `style.css` | الستايلات |
| `config.js` | البيانات والثوابت |
| `app.js` | منطق التطبيق + PWA |
| `sw.js` | Service Worker |
| `manifest.json` | PWA manifest |
| `icon-192.png` | أيقونة التطبيق |
| `icon-512.png` | أيقونة التطبيق كبيرة |

## PWA

الموقع قابل للتثبيت كتطبيق على الموبايل.
